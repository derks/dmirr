dMirr version ${version}

dMirr is a distributed mirror server manager. It's purpose is to organize data for upstream projects needing global mirror servers, and the generous sites willing to help mirror those projects.

Features

  • Yum "mirror list" interface. Closest match ordered via geoip location
  • RESTful web framework using Apache, WSGI, and TurboGears 2

Support

  • http://github.com/derks/dmirr/issues
